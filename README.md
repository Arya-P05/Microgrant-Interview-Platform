# VC Interview Booking

Booking app for Round 1 and Round 2 interviews of Techyon Microgrants. Lol built this in 2 nights so lmk if anything breaks ever.

The app opens on Round 2 by default since that was what we last needed but update to round 1 when next year comes back around (`/` redirects to `/round2`).

## Quick start

```bash
cd next-app
cp .env.local.example .env.local
# send me a dm for the keys if you need them or sign in to the one signal and use the admin supabase account
npm install
npm run dev
```
